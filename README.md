# SignTry.com

**SignTry.com** helps users create professional email signatures in minutes.  
Choose a template, customize your details, and save your personalized signature effortlessly.

## Structure

- **/app** – Next.js dashboard (app.signtry.com)
- **/landing** – Marketing landing page (signtry.com)

## Tech Stack

Next.js · React · TypeScript · Tailwind CSS · Framer Motion · Supabase · Stripe

## Development

```bash
npm install
npm run dev
```
